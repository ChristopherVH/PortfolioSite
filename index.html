Nimbus Playlist

Platform for playing music and creating playlists

Nimbus Playlist is a single page app that I built using Flux architecture and React.js for the views. I also implemented fuzzy search that limits amount of database queries to speed up searches. I used the Wave Surfer library to enhance user experience by displaying colorful sound bars upon pressing play.

Don't Push Me

Javascript front-end

Don't Push Me utlizes Canvas, an HTML element, to animate the moving blocks and to dynamically update the position of the cute little penguin. I learned Google's Firebase API to create a real time database to display the top 10 highest scores. I wrote Javascript logic so collisions, block stacking, and movement function intuitively.

Chess

Ruby back-end

The source code can be found here! A fully functional in terminal chess game with AI, written entirely in Ruby. Built using elements of inheritance, encapsulation, and polymorphism to keep code DRY, also implemented error handling to prevent moves that aren't allowed in a real chess game.
